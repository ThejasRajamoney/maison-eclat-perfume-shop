# Maison Éclat — Cinematic Perfume Shop

A cinematic luxury perfume shop website built with **HTML, CSS, JavaScript, GSAP, and ScrollTrigger**.  
The site is designed like a premium fragrance brand landing page with smooth scroll animations, product showcases, interactive hover effects, responsive layouts, and a dark luxury visual style.

![Maison Éclat Preview](./preview.png)

---

## ✨ Overview

**Maison Éclat** is a fictional high-end perfume shop website focused on visual storytelling, motion design, and luxury branding.

The goal of this project is to create a portfolio-level frontend experience using only vanilla web technologies and GSAP animations, without relying on React or a full framework.

---

## 🚀 Features

- Cinematic loading screen
- Luxury dark fragrance-brand design
- Animated hero section with custom CSS perfume bottle
- GSAP-powered reveal animations
- ScrollTrigger-based scroll effects
- Interactive product cards
- Horizontal product showcase on desktop
- Mobile-friendly stacked product layout
- Smooth responsive design for all devices
- Custom cursor on desktop
- Magnetic hover effects
- Accordion FAQ section
- Newsletter form interaction
- Add-to-bag toast notification
- Performance optimizations for mobile and low-power devices
